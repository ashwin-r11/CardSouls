## 🟠 Reddit

### r/webdev / r/nextjs / r/sideproject

**Title:** I built an AI collectible card generator for fictional characters — $0/month, fully open source

**Body:**

Hey everyone! I just shipped **CardSouls** — it turns your emotional connection to a fictional character into an aesthetic Polaroid-style collectible card.

**What it does:**
- You input a character name + what they mean to you emotionally
- AI generates a custom portrait, poetic verse, color palette, and matching song
- You get a downloadable PNG card

**The stack (all free):**
- Next.js 16 + TypeScript
- Groq (Llama 3.3 70B) — 14,400 free requests/day
- Pollinations.ai for image generation — unlimited, no API key
- Deezer for music — free, no auth
- Vercel for hosting

**The journey to get here was brutal:**
- Started with Claude's API (5 separate agents) — way too expensive
- Moved to Gemini — hit the 20 req/day free tier wall
- Gemini Pro? 0 free tier requests 💀
- Finally landed on Groq + collapsed 5 agents into 1 single call

I documented everything I learned in the repo. Would love feedback!

🔗 **GitHub:** github.com/ashwin-r11/CardSouls
